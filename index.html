Como criar os dois flows no Power Automate

Passo a passo — ~15 minutos, sem código
📋 Pré-requisito: criar uma Lista no SharePoint 1. Abra o SharePoint da O-I → seu site de equipe PCM
2. Clique em + Novo → Lista
3. Nome: ArvorePCM
4. Adicione uma coluna do tipo Várias linhas de texto com nome Conteudo
5. Crie um item chamado arvore_5010 nessa lista (deixe Conteudo vazio)
▼ FLOW 1 — Carregar (GET) 1. Power Automate → + Criar → Fluxo de nuvem instantâneo
2. Trigger: "Quando uma solicitação HTTP é recebida"
3. Adicionar ação: SharePoint → Obter item
  · Endereço do Site: seu site SharePoint
  · Nome da Lista: ArvorePCM
  · ID: 1 (o item que você criou)
4. Adicionar ação: Resposta
  · Código de Status: 200
  · Corpo: @{outputs('Obter_item')?['body/Conteudo']}
  · Cabeçalhos: Content-Type: application/json
5. Salvar → copie a URL HTTP POST gerada → cole no campo GET deste app
▲ FLOW 2 — Salvar (POST) 1. Power Automate → + Criar → Fluxo de nuvem instantâneo
2. Trigger: "Quando uma solicitação HTTP é recebida"
  · Esquema JSON do Corpo: {"type":"object","properties":{"payload":{"type":"string"}}}
3. Adicionar ação: SharePoint → Atualizar item
  · Endereço do Site: seu site SharePoint
  · Nome da Lista: ArvorePCM
  · ID: 1
  · Conteudo: @{triggerBody()?['payload']}
4. Adicionar ação: Resposta
  · Código de Status: 200
  · Corpo: {"ok":true}
5. Salvar → copie a URL HTTP POST gerada → cole no campo POST deste app
⚠ Importante: Os flows precisam estar Ativos e com permissão de execução para toda a equipe (compartilhe o flow com os membros da equipe PCM no Power Automate).

⚠ Conflito detectado

O SharePoint tem uma versão mais recente
O que fazer? O SharePoint foi atualizado por outro membro da equipe enquanto você trabalhava.
